# 변경 로그

## [1.0.0] - 2026-02-27

### 추가
- 로컬 전용 암호화 메모 애플리케이션 첫 릴리즈
- AES-256 암호화로 모든 메모 보호
- 실시간 자동 저장 (1초 지연)
- 음성 녹음 및 재생 기능
- Gemini AI 통합 (음성→텍스트, 요약)
- Word 문서 내보내기
- 비밀번호 보호 기능
- 다중 비밀번호 지원 및 통합
- 다크 테마 UI
- 자동 업데이트 시스템


### 기술 스택
- .NET 10.0 / WPF
- NAudio (음성 녹음)
- DocumentFormat.OpenXml (Word 내보내기)
- QRCoder (QR 코드 생성)
- Newtonsoft.Json (JSON 처리)
